# TrustPay - Demo App

This is a functional React prototype of the TrustPay application, designed with a sleek mobile-first UI wrapper. 

## Features
- **Global State Management:** Seamless synchronization between Lend, Borrow, Split, and Home screens.
- **Lend & Borrow:** Send and request money instantly, with auto-pay tenure tracking and realistic success flows.
- **Split Bills:** Highly interactive split group creator supporting equal, percentage, and exact amount division types.
- **Scan & Pay:** Verify UPI inputs with a live camera scanning animation.

## Setup
To run the app locally:
```bash
npm install
npm run dev
```
